# OBJ

Simple OBJ (wavefront model format) file utils

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'obj'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install obj

## Usage

    obj-parse <file_path>
